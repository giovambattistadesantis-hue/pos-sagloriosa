☁️ Configurazione Firebase

Inserisci le credenziali del tuo progetto Firebase per sincronizzare catalogo e casse in tempo reale.
Visita console.firebase.google.com → Impostazioni progetto → Web app.

🔑 Credenziali Firebase

📱 Istruzioni rapide

1. Crea progetto Firebase → Abilita Firestore Database (modalità produzione)
2. In Firestore → Regole → incolla: allow read, write: if true;
3. Impostazioni progetto → Web app → copia i dati qui sopra
4. Ogni cassa apre pos_cassa.html e preme ☁️ per inserire le stesse credenziali + nome cassa (es. "Cassa 1")
